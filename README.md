# Android Weather Application

Java ile geliştirilmiş basit bir Android hava durumu uygulamasıdır. Bu proje, Android uygulama geliştirme pratiği kazanmak ve mobil tarafta proje yapısını, ekran akışını ve uygulama sürümleme süreçlerini deneyimlemek amacıyla oluşturulmuştur.

## Proje Hakkında

Android Weather Application, kullanıcıya hava durumu bilgilerini mobil arayüz üzerinden sunmayı amaçlayan bir Android uygulamasıdır. Proje, Java tabanlı Android geliştirme sürecine odaklanır ve temel uygulama mimarisi ile Android proje yapısını içerir.

## Kullanılan Teknolojiler

- Java
- Android Studio
- Gradle

## Özellikler

- Android tabanlı mobil uygulama yapısı
- Java ile geliştirilmiş uygulama bileşenleri
- Gradle tabanlı proje yapılandırması
- Sürümleme ve release yönetimi

## Kurulum

1. Bu repoyu klonlayın:
   ```bash
   git clone https://github.com/byFischer/Android-WeatherApplication.git
   ```

2. Projeyi Android Studio ile açın.

3. Gerekli Gradle bağımlılıklarının yüklenmesini bekleyin.

4. Emülatör veya fiziksel cihaz üzerinde projeyi çalıştırın.
